使用指南

下面您只需要在公众号和博客中做如下设置，然后就等着来自博客的自然增长吧！

第一步：公众号设置

登录要增粉的微信公众号，按如下规则设置关键词回复

  • 关键词，设置为：vip
  • 回复内容，设置为如下文本内容：
<a href="https://my.openwrite.cn/code/generate?blogId=21839-1591198660133-778">点击该链接，获取博客解锁验证码</a>

第二步：博客设置

在您博客的文章页面的末尾，增加如下js代码。

<script src="https://my.openwrite.cn/js/readmore.js" type="text/javascript"></script>
<script>
    document.getElementsByClassName('article-container')[0].setAttribute("id","container")
    const btw = new BTWPlugin();
    btw.init({
        id: 'container',
        blogId: '21839-1603629348090-282',
        name: '努力升级中',
        qrcode: 'https://i.loli.net/2020/10/25/fVQpdYESFXjGBmo.png',
        keyword: '验证码',
    });
</script>

其中id中设置的container需要用户根据文章页面中的文章容器来调整，或者直接将文章最外面的容器设置为该id。
