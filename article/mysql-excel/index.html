Mysql 导出Excel表

October 22, 2020

命令

mysql -h x.x.x.x -P 3306 -u用户名 -p密码 -e "select * from test" 数据库 > /tmp/test.xls

说明

-h ：数据库Host

-P： 数据库端口

-u： 用户名

-p： 密码

-e：sql语句

/tmp/test.xls： 文件名
